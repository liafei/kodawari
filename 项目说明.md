# 项目说明

`kodawari.cli.main`（主 CLI 入口）

本仓库使用 `scripts\kodawari.ps1` 作为 repo-local canonical entrypoint。
首次本地运行可执行 `scripts\bootstrap_kodawari.ps1`，之后优先使用：

- `.\scripts\kodawari.ps1 gate --help`
- `.\scripts\kodawari.ps1 telemetry --help`

稳定性 lane 固定入口：

- `scripts\run_always_on_lane.ps1`
- `scripts\run_integration_lane.ps1`
- `scripts/test_lane_recipes.json`
- `kodawari lane-history-fetch`

CI / standing proof 产物：

- `.github/workflows/kodawari-always-on.yml`
- `.github/workflows/kodawari-integration.yml`
- `.github/workflows/kodawari-standing-proof.yml`
- `lane_stability_always-on.json`
- `lane_stability_integration.json`
- `lane_triage_always-on.json`
- `lane_triage_integration.json`
- `lane_history_manifest.json`
- `lane_weekly_trend.json`
- `lane_weekly_trend.md`
